SPZ - Stochastisches Probabilistisches Verfahren

1. Funktionsweise

Das SPZ ist eine Variante des Gale-Shapley-Algorithmus. Der Ablauf:

  1. Normalisierung: Für jeden Schüler werden die Stimmen so skaliert, dass ihre Summe 1 ergibt. Aus 10, 6, 3 wird also 0,526, 0,316, 0,158.
  2. Projektauswahl: In jeder Runde wird das Projekt mit den meisten verbleibenden Stimmen ausgewählt.
  3. Zuordnung: Per Zufall wird ein Schüler aus diesem Projekt ausgewählt, basierend auf den normalisierten Werten, wer dabei einen höheren Wert hat erhält größere Chancen, den Platz zu erhalten .
  4. Vergleich: Wenn der ausgewählte Schüler bereits ein Projekt hat, wird geprüft, ob das neue Projekt besser ist. Falls ja, findet ein Tausch statt.
  5. Wiederholung: Der Vorgang wiederholt sich, bis alle Schüler zugeordnet sind oder keine freien Plätze mehr verfügbar sind.

Charakteristik: Stochastisch - bei mehreren Durchläufen können unterschiedliche Ergebnisse entstehen.

2. Anwendungssituationen

Das SPZ eignet sich besonders für:

  • Fälle, in denen eine ausgewogene Verteilung erwünscht ist
  • Situationen, in denen kein Projekt priorisiert werden soll
  • Mehrere unabhängige Durchläufe mit unterschiedlichen Ergebnissen
  • Wenn die Zufallskomponente erwünscht ist

3. Datenstruktur der Eingaben

Das SPZ erwartet zwei CSV-Dateien:

Projekt-Kapazitäten (projekt.csv)

Projekt,Kapazitaet
Projekt A,35
Projekt B,28
Projekt C,21
...

Spalten: Projektname, Kapazität (Anzahl verfügbarer Plätze)

Hinweis: Projektnamen dürfen Kommas enthalten - diese werden korrekt verarbeitet.

Schüler-Wahlen (schueler.csv)

Vorname,Nachname,class,Projekt A,Projekt B,Projekt C,...
Max,Mustermann,7a,10,0,0,1,0,0,...
Anna,Schmidt,7b,5,6,0,0,0,0,...
...

Spalten: Vorname, Nachname, Klasse, dann für jedes Projekt eine Spalte mit der Stimmenanzahl

4. Verwendungsanleitung

So verwenden Sie das SPZ:

  1. Stellen Sie sicher, dass keine Felder leer sind, sondern mindestens eine 0 enthalten
  2. Laden Sie die Projekt-Kapazitäten-Datei hoch
  3. Laden Sie die Schüler-Wahlen-Datei hoch
  4. Klicken Sie auf "Auswerten"
  5. Die CSV-Datei wird automatisch heruntergeladen

Tipp: Öffnen Sie die Browser-Konsole (F12) für detaillierte Analyseergebnisse.

ILP - Ganzzahlige Optimierung

1. Funktionsweise

Das ILP-Verfahren verwendet mathematische Optimierung, um die bestmögliche Zuordnung zu finden:

  1. Modellierung: Das Problem wird als Ganzzahlige Lineare Optimierung (ILP) formuliert.
  2. Zielfunktion: Maximierung der Gesamtzufriedenheit basierend auf den Präferenzen.
  3. Randbedingungen: Jeder Schüler erhält maximal ein Projekt; jedes Projekt hat eine maximale Kapazität.
  4. Lösung: Ein Solver berechnet die optimale Zuordnung.

Charakteristik: Deterministisch - bei gleichen Eingabedaten kommt immer das gleiche Ergebnis.

Bald verfügbar: Dieses Verfahren wird in Kürze implementiert sein.

2. Anwendungssituationen

Das ILP-Verfahren eignet sich besonders für:

  • Konsistente Ergebnisse bei wiederholten Durchläufen
  • Maximierung der Gesamtzufriedenheit
  • Fälle, in denen Reproduzierbarkeit wichtig ist
  • Wenn die bestmögliche Lösung garantiert werden soll

3. Datenstruktur der Eingaben

Das ILP-Verfahren erwartet zwei CSV-Dateien:

Projekt-Kapazitäten (projekt.csv)

Projekt,Kapazitaet
Projekt A,35
Projekt B,28
Projekt C,21
...

Spalten: Projektname, Kapazität (Anzahl verfügbarer Plätze)

Schüler-Wahlen (schueler.csv)

Vorname,Nachname,class,Projekt A,Projekt B,Projekt C,...
Max,Mustermann,7a,1,2,3,0,0,0,...
Anna,Schmidt,7b,2,1,3,0,0,0,...
...

Spalten: Vorname, Nachname, Klasse, dann für jedes Projekt eine Spalte mit der Wunschnummer

Wertesystem: 1 = Erstwunsch, 2 = Zweitwunsch, 3 = Drittwunsch, 0 = keine Präferenz

Wichtig: Jeder Schüler hat genau eine 1, eine 2 und eine 3 in seiner Zeile (oder 0 wenn kein Wunsch)

4. Verwendungsanleitung

So verwenden Sie das ILP-Verfahren:

  1. Stellen Sie sicher, dass beide CSV-Dateien im UTF-8 Format vorliegen
  2. Die Schüler-Wahlen-Datei verwendet das Wertesystem 1, 2, 3 statt 10, 6, 3
  3. Laden Sie die Projekt-Kapazitäten-Datei hoch
  4. Laden Sie die Schüler-Wahlen-Datei hoch
  5. Klicken Sie auf "Auswerten"
  6. Die CSV-Datei wird automatisch heruntergeladen

Tipps

  • Stellen Sie sicher, dass genügend Gesamtkapazität vorhanden ist für alle Schüler
  • Bei sehr beliebten Projekten kann es sein, dass nicht alle Erstwünsche erfüllt werden können
  • Öffnen Sie die Browser-Konsole (F12) für detaillierte Analyseergebnisse
  • Für SPZ verwenden Sie das Punktesystem, Punkte spiegeln Präferenzen dar, für den ILP Algorithmus brauchen sie reguläre Erst-, Zweit-, und Drittwünsche (1, 2, 3)
